# 🐆 Jagüi: Mi Primer Copiloto - ¡Aprende Inteligencia Artificial Jugando! 🤖⚡

¡Bienvenido a **"Jagüi: Mi Primer Copiloto"**, la aventura educativa infantil definitiva (estilo Duolingo) para enseñar los fundamentos y secretos de la Inteligencia Artificial a niños de **8 a 12 años**! 

Guiados por **Jagüi**, un adorable jaguar tecnológico 3D con collar de circuitos cian, los niños explorarán mapas selváticos cibernéticos, entrenarán modelos de Machine Learning por proximidad de centroides en un laboratorio interactivo 2D, conversarán con la IA nativa espacial de Gemini y desbloquearán medallas premium en una experiencia totalmente inmersiva y gamificada de nivel comercial 10+/10.

👉 **Despliegue en Vivo:** [https://jagui-ia.vercel.app](https://jagui-ia.vercel.app)
👉 **Código Fuente:** [https://github.com/jovfranco-tech/jag-i-mi-primer-copiloto](https://github.com/jovfranco-tech/jag-i-mi-primer-copiloto)

## 💎 Características Premium 10+/10 Implementadas

### 🐆 1. Arte Tridimensional de Google Imagen
Aprovechamos al máximo la potencia de los modelos generativos de Google Imagen para diseñar renders 3D premium de nivel de producción de animación de Hollywood:
*   **Jagüi Héroe (`assets/jagui_mascot_hero.png`)**: La portada y el perfil muestran un adorable jaguar bebé 3D estilo Pixar en la ciber-selva cibernética.
*   **Jagüi Científico (`assets/jagui_scientist.png`)**: Vestido con gabardina de laboratorio y gafas holográficas VR neón, guiando de forma lúdica a los niños en sus experimentos del Lab.
*   **Medalla de Logro (`assets/achievement_badge.png`)**: Una deslumbrante medalla de oro 3D con circuitos impresos y hojas selváticas neón que sustituye a los emojis planos en las recompensas de perfil con destellos metálicos (`shinyGold`).
*   **Fondo Selva Neón (`assets/cyber_jungle_bg.png`)**: Un fondo de pantalla de selva cibernética oscura, cubierto de lianas neón y circuitos fluorescentes, integrando una estética de videojuego fascinante.

### 🎁 2. Gamificación Extrema: Misiones Diarias y Cofre de la Selva
*   **Misiones Diarias 🗺️**: Un panel flotante y adaptativo en el mapa de niveles que reta al niño a cumplir tres objetivos diarios interactivos: entrenar a la IA en el Lab, platicar con Jagüi, o conseguir 50 XP. Las misiones se tachan automáticamente con visuales de éxito (`✅`).
*   **Cofre de la Selva 🎁**: Un cofre misterioso interactivo que se balancea elásticamente al completarse las misiones o acumular XP. Al abrirse, reproduce una fanfarria arpegiada de Web Audio, animaciones triples de confetti, y otorga un premio aleatorio de `+50 XP`.

### ⚡ 3. Sonido Retro-Arcade Whimsical (Web Audio API)
Hemos diseñado un motor de síntesis de sonido analógico directo en el navegador usando osciladores puros y envolventes de ganancia exponencial en Web Audio API:
*   `'powerup'`: Un sweep de frecuencia de diente de sierra ascendente futurista para la activación del **Modo Súper Cerebro 🧠⚡**.
*   `'chest'`: Un arpegio brillante ascendente y alegre al abrir cofres.
*   `'bubble'`: Un pop acuoso corto y táctil para las burbujas de diálogo del Chatbot.
*   `'equip'`: Un sonido cyber de confirmación en la tienda de Jagüi.
*   Controles integrales de volumen analógico en el Perfil para la música selvática de fondo y los efectos de sonido (SFX).

### 🛡️ 4. Privacidad y Seguridad Parental (Kids Safety First)
*   **Guía para Padres y Educadores**: Inyección de una tarjeta colapsable en el Chatbot Gemini detallando los estrictos criterios de seguridad seguidos (privacidad absoluta sin capturar información sensible, filtros éticos para respuestas amigables e indicaciones para un uso seguro de la IA como copiloto).

### 📱 5. PWA Mobile-First Ultra-Fluida
*   Adaptación de diseño elástico compatible con tabletas y móviles (targets táctiles de mínimo 48px).
*   Manifiesto PWA autoinstalable y soporte Service Worker offline instantáneo para jugar sin fricciones en cualquier parte de la selva.

## 🛠️ Stack Tecnológico
*   **Core:** Vanilla HTML5, CSS3 Custom Properties (CSS Grid, Variables, Keyframe Animations, Backdrop Filters), Javascript Puro (PWA, Service Worker).
*   **Bases de Datos & Cloud:** Firebase Auth (inicios de sesión del Club de Científicos) y Firestore Cloud Sync.
*   **Audio Engine:** Web Audio API (Sintetizadores analógicos integrados).
*   **AI Engine:** Google Gemini Pro API (interacciones del Chat de Jagüi).

## 🚀 Instrucciones de Ejecución Local

Para probar o ejecutar la aplicación de forma local en tu computadora:

1.  Clona este repositorio de GitHub:
    ```bash
    git clone https://github.com/jovfranco-tech/jag-i-mi-primer-copiloto.git
    cd jag-i-mi-primer-copiloto
    ```
2.  Levanta un servidor de desarrollo local (por ejemplo usando VS Code Live Server, npm http-server o python):
    ```bash
    # Usando Python
    python -m http.server 8000
    ```
3.  Abre `http://localhost:8000` en tu navegador web. ¡Y que comience la aventura selvática! 🐆🚀
